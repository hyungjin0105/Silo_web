## 핵심 결정사항 요약

- 사용자 페이지 = **모바일 웹** (QR 진입). 네이티브 앱 X.
- 공실 페이지 = **PG 연동 포함**. 멤버십·지역화폐 2026 미포함.
- 데이터 = **역설계** + **3막 (Before/During/After)** + **4태그 이벤트 표준** (누가·언제·어디서·뭘).
- 임대인 = 건물주 / 임차인 = 팝업 운영자 (이 프로젝트에서 고정 정의).
- 다국가 골격 = 처음부터 변수화. KO 1차, EN/IT 추가.
- 여름까지 풀스택 1차 완성. 9-10월 "다시 육거리" XL 본 검증.
- 세 페이지가 **하나의 이벤트 로그**를 공유 — 사용자 페이지의 QR 스캔이 공실 페이지 임차인 대시보드의 방문자, 데이터 시각화의 유동인구가 됨.
