# 02. 사용자 페이지 — 기능 기획

> **대상**: 행사 참여자(시민·방문객). 남녀노소.
> **형태**: QR로 진입하는 **모바일 웹**. 설치 불필요. 비회원으로도 핵심 기능 사용 가능.
> **목적**: 행사·장소·먹거리·콘텐츠를 한눈에 + 이용 행동이 자연스럽게 데이터(D1/D2/D6/D7)가 되도록 설계.

---

## 1. 사용자 정의·페르소나

### 1.1 1차 페르소나 (사용 빈도 높음)
- **P1. 포항 주민, 20~50대** — 모바일에 익숙. 가족과 함께 또는 친구와 행사 참여.
- **P2. 인근 도시 방문객, 20~40대** — 영화제·페스티벌 보러 옴. 외지인 관점에서 정보 탐색.

### 1.2 2차 페르소나 (접근성 핵심)
- **P3. 어린이 동반 가족** — 키즈 친화 정보 필요. 부모가 대신 조작.
- **P4. 어르신 (60+)** — 큰 글씨·간단한 동선. 복잡한 회원가입 부담.
- **P5. 외국인 관광객** — 영어 우선, 길찾기·결제·먹거리 핵심.

### 1.3 사용 빈도 모델
- 거리 통행 중 우연 발견 → QR 스캔 → 첫 진입 (가장 많은 진입)
- 행사 전 사전 정보 탐색 (포털·SNS 링크 유입)
- 행사 중 현장 탐색·동선 (재진입)
- 행사 후 후기·기억 회상 (재진입 1~2주)

---

## 2. 사용 시나리오 (대표 3개)

**S1. "지나가다 들른 시민"**
QR 발견 → 모바일 웹 오픈 → 오늘 진행 중 행사 카드 보임 → 가장 가까운 팝업 클릭 → 길찾기 시작 → 도착 후 입구 QR 스캔 → 스탬프 1개 적립 → 후기 별점 → 나옴.

**S2. "사전에 계획하고 방문한 외지인"**
검색 또는 SNS 링크로 진입 → 행사 캘린더 확인 → 영화제 예매 → 야시장 추천 코스 확인 → 방문일 알림 받음 → 현장에서 코스 따라 이동 → 각 지점 QR 스캔 → 스탬프 완성 → 회고 페이지에 사진 자동 모임.

**S3. "어르신 가족과 함께 방문"**
자녀가 대신 화면 조작 → 큰 글씨 모드 ON → 휠체어 접근 가능 동선 필터 → 추천 코스 → 짧고 단순한 동선 → QR 스캔 → 끝.

## 4. 정보 구조 (IA)

```
[홈 (A1)]
 ├─ [지금 진행 중 행사 카드들 (A2/A3 미리보기)]
 ├─ [현재 위치 기반 추천 (B3)]
 ├─ [오늘의 추천 코스 (A8)]
 └─ [로컬 스토리 카드 (A7)]

[탭바 / 글로벌 네비게이션]
 ├─ 홈 (A1)
 ├─ 지도 (B1)
 ├─ 행사 (A2 → A3)
 ├─ 팝업·먹거리 (A4·A6 → A5)
 ├─ 스토리 (A7)
 └─ 내 정보 (D5/D7 — 비로그인 시 로그인 유도)

[QR 진입 페이지 — 컨텍스트 의존]
 ├─ 거리 zone QR → 홈으로 (방문 카운트만)
 ├─ 팝업 QR → 해당 팝업 상세 (A5)
 ├─ 행사장 QR → 해당 행사 상세 (A3) + 체크인
 ├─ 스토리 QR → 해당 스토리 상세 (A7)
 └─ 스탬프 QR → 스탬프 적립 + 다음 안내

[독립 기능 — Floating / 컨텍스트 진입]
 ├─ 검색 (A9)
 ├─ QR 스캔 (C2)
 ├─ 후기 작성 (C8)
 └─ 공유 (C6)
```

## 7. 화면 구성 우선순위 (와이어프레임 작업 시 순서)

1. **QR 진입 ~ 홈** (A1) — 첫 인상이자 가장 빈번한 진입
2. **행사 상세 (A3) / 팝업 상세 (A5)** — 콘텐츠 페이지의 표준 템플릿
3. **지도 (B1)** — 위치 기반 탐색의 핵심
4. **후기·별점 (C8)** — 데이터 수집의 핵심
5. **스탬프 / 미션 (C3·C4)** — 재방문·동선 유도
6. **마이페이지 (D5)** — 회원 진입의 보상 면

## 8. 톤·디자인 방향 (메모 — design_direction.md 와 정합)

- 사진 중심 편집 디자인. 행사·팝업·로컬스토리가 콘텐츠의 90%.
- 파랑은 액센트로만 사용. 메인은 사진의 자연색.
- 장식 모션은 배제. 정보 전달 우선.
- "어플리케이션 형태"라고 했어도 네이티브 앱 흉내는 내지 않음 — 모바일 웹 답게 가볍게.
- 단순 모드(E2)는 별도 디자인 변주 필요. 기본 모드에서 토글로 진입.
