# Popin — 디자인 무드보드

> **버전** v0.1 · **작성일** 2026-05-19
> 사용자가 전달한 레퍼런스 10장 + 명시 의도 ("Nexora 톤 미니멀 애니메이션 + 헤로는 사진 X, 타이포 모션 그래픽")를 기반.
> 이 문서가 디자인 토큰·마이크로카피 작업의 기반.

---

## 1. 레퍼런스 10장 분석

각 이미지에서 **추출할 패턴**과 **추출하지 않을 부분**을 명시.

### 1.1 Popin 자체 시안 (이미지 1)
어두운 동네 풍경 헤로 + 큰 흰 한글 타이포 "당신의 아이디어가 머물 자리." + 파랑·회색 CTA + 헤더에 KO/EN 토글·로그인·회원가입.

- **추출 ○** — 큰 한글 헤로 카피 톤("당신의 아이디어가 머물 자리"), 헤더 단순함, "지금 매장이 열리는 동네" 하단 칩 라인, 헤더 우측 KO/EN 토글 + 검정 회원가입 버튼 패턴.
- **추출 ✕** — 동네 풍경 사진 헤로 → **사용자가 명시적으로 거절**. 대체 = 타이포 모션 그래픽.
- **의문점** — 이게 본인이 이미 만든 시안인지, 영감용인지? **결정 #1 필요.**

### 1.2 Forma AI Pricing (이미지 2)
다크 배경 + 글래스모피즘 가격 카드 3개 + 배경에 거대 백드롭 "Pricing" 텍스트 + 푸른 글로우.

- **추출 ○** — 거대 백드롭 텍스트가 카드 뒤에 겹치는 레이어드 구성. 이 패턴은 헤로 모션 그래픽에 활용 가능.
- **추출 ✕** — 다크 모드·글래스 효과·SaaS틱 글로우. Popin은 다크 모드가 메인 아님(타운 페이지·매물 사진은 밝아야).

### 1.3 Nexora "Curiosity-led tools" (이미지 3) ★ 사용자 강조
다크 배경 + 오렌지/블루 색면 카드 비대칭 그리드 + 이미지·일러스트·텍스트 카드 혼합 + 큰 라운드 + "01/" "02/" 같은 작은 인덱스 라벨.

- **추출 ○** — **이게 핵심 톤**.
  - 비대칭 그리드 (다양한 크기 카드를 한 줄에)
  - 큰 라운드 (estimated 24~32px)
  - "01/" "02/" 인덱스 라벨 — 매거진 감성
  - 오렌지·블루·다크 + 일러스트 톤의 그래픽 영상
  - 짧고 강한 카피 ("Where Knowledge Begins", "Great Questions Unearth Hidden Gems") — 동사·시적 톤
- **응용** — Popin 헤로를 이렇게: "당신의 아이디어가 머물 자리" 같은 큰 카피 + 오렌지 색면 + 작은 일러스트/타이포 모션 그래픽 + "01/ 동네 1층" 같은 인덱스 라벨.

### 1.4 향수 광고 (이미지 4)
파랑 색면 + 핑크 색면 + 큰 인물 사진 + 작은 비대칭 제품 사진.

- **추출 ○** — **컬러 블록 분할 레이아웃**. 한 섹션을 두 색면으로 나누고 사진은 작게. 타운 페이지에서 활용 가능 (한 색면에 동네 카피, 다른 색면에 매물 사진).
- **추출 ✕** — 파랑+핑크 조합 자체는 Popin에 안 맞음(부동산보다 뷰티 톤).

### 1.5 Architecture posters / FORMAEU (이미지 5)
오렌지·검정·크림 색면 + 굵은 산세리프 타이포 ("Rénovation & Réhabilitation", "Architecture Durable") + 건축 일러스트.

- **추출 ○** — **이미지 3과 함께 핵심 톤**.
  - 오렌지·크림·검정 3색만으로 강한 인상
  - 굵은 산세리프 (제목 무게가 페이지를 잡음)
  - 건축적·도시적 모티프 — Popin이 1층 상가 서비스니까 결이 맞음
  - 포스터·매거진 톤
- **응용** — 타운 페이지를 포스터 시리즈처럼: 각 타운 = 한 포스터. "성수동 팝업타운"을 큰 타이포로, 작은 동네 일러스트를 곁들임.

### 1.6 MIMCO Capital (이미지 6)
미니멀 헤로 + 건물 사진 + 거대 흰 로고 "MIMCO" + 우측 작은 녹색 원 CTA.

- **추출 ○** — 헤더 단순함, 거대 로고를 헤로에 겹치는 패턴, 작은 강조 색(녹색)을 한 점에만 쓰는 절제.
- **추출 ✕** — 건물 사진 헤로(사용자 거절).
- **응용** — 큰 "Popin." 워드마크를 헤로 안에 또는 푸터에 거대하게 한 번 보여주는 패턴.

### 1.7 Appear Here (이미지 7)
풀스크린 영상 헤로 + 가운데 흰 큰 타이포 "FIND SPACE FOR YOUR IDEA" + 검색바 + 트러스트 로고 (Apple·Nike·Chanel·Supreme·adidas·Netflix·Spotify).

- **추출 ○** — 검색바 패턴(지역+날짜+기간 가로 한 줄), 트러스트 로고 라인, 헤로 카피 톤 ("FIND SPACE FOR YOUR IDEA" = 우리의 "당신의 아이디어가 머물 자리").
- **추출 ✕** — 영상 헤로(사용자 거절). 영문 대문자 일변도 (한글 대비 약함).

### 1.8 VEX "Shaping tomorrow..." (이미지 8)
도시 사진 + 좌하단 흰 큰 타이포 + 우하단 짧은 태그 라인 + 둥근 모서리 큰 컨테이너 헤더.

- **추출 ○** — 카피 좌하단 정렬 패턴, 둥근 헤더(pill) — 헤더가 화면에서 떠 있는 듯한 느낌.
- **추출 ✕** — 도시 풍경 사진 헤로(사용자 거절).

### 1.9 "TRACK NUMBER 09" 포스터 (이미지 9)
큰 오렌지 색면(상단) + 크림 색면(하단) + 거대 숫자 "09" + 작은 라벨 "CONCEPT POSTER SERIES" + 인물 사진을 색면 경계에 걸침.

- **추출 ○** — **이미지 3·5와 동일 결**. 색면 2분할 + 거대 숫자 + 인물 작게 + 작은 캡션. 이 톤이 사용자가 좋아한다는 신호 명확함.
- **응용** — 타운 페이지 헤로: 상단 색면(오렌지 또는 크림)에 거대 "01/" 인덱스 또는 거대 한글 "성수" + 하단 색면에 매물 정보.

### 1.10 SE7EN 포스터 (이미지 10)
크림 배경 + 거대 빨강 타이포 "SE7EN" + 인물 사진을 타이포 안에 겹침 + 작은 모노타입 캡션 + 일본어/영어 혼용 캡션.

- **추출 ○** — **이미지 3·5·9와 동일 결**. 거대 타이포에 사진을 끼워 넣는 레이어드 패턴, 모노타입(typewriter) 작은 캡션, 다국어 혼용.
- **응용** — 타운 페이지·헤로에 한국어 큰 타이포 + 영문 작은 모노 캡션 ("THE TOWN NO. 01").

## 2. 추출된 디자인 원칙 (7가지)

레퍼런스 10장에서 도출. 이게 토큰·컴포넌트 결정의 헌법.

### P1. 거대 타이포가 페이지를 잡는다
이미지 3·5·8·9·10 공통. **헤로의 주인공은 큰 한글 타이포** (사진·일러스트는 보조). 무게 700~800대 산세리프. 데스크탑에서는 viewport의 1/3 ~ 1/2을 타이포가 차지.

### P2. 색면 분할 + 그래픽 디자인 톤
이미지 4·5·9 공통. **사진·정보를 색면 위에 얹는다**. 두 색면을 한 화면에서 분할(상하 또는 좌우)하는 패턴이 핵심. 솔리드 컬러 면적이 크다.

### P3. 제한된 팔레트 + 강한 액센트 한 색
이미지 5·9 = 오렌지/크림/검정. 이미지 6 = 흰/회색 + 녹색 한 점. 이미지 10 = 크림/검정 + 빨강 한 색. **메인 3색 + 액센트 1색** 패턴. Popin도 이 절제 따라야 generic AI 디자인 안 됨.

### P4. 매거진·포스터 감성
이미지 3·5·9·10 공통. "01/" "TRACK NUMBER" "DESIGN NO. 45" 같은 **작은 인덱스 라벨이 큰 타이포 옆에 곁들임**. 모노타입(typewriter) 폰트로 작게. 이게 편집·포스터 톤의 핵심 시그널.

### P5. 비대칭 그리드
이미지 3 핵심. **카드 크기가 다 다름**. 한 행에 큰 카드 1 + 작은 카드 1·2. 모든 카드가 같은 크기면 AI틱.

### P6. 헤더는 가볍게
이미지 6·8 공통. 헤더는 흰 배경 또는 떠 있는 둥근 컨테이너. 메뉴 항목 4~5개 이하. **Popin 헤더는 미니멀 유지**.

### P7. 한글·영문 혼용 + 모노타입 캡션
이미지 10. 한글이 큰 타이포로 영문/일본어가 작은 모노타입 캡션으로 곁들임. **Popin은 한국 서비스라 한글 큰 타이포 + 영문 모노 캡션 ("POPUP TOWN NO. 01") 조합** 권장.

## 3. 사용자 명시 의도 (강조)

> "특히 색감 톤 분위기 3의 이미지처럼 미니멀 애니메이션 느낌으로 웹사이트를 브랜딩하고 싶어. hero 섹션에도 실물 사진이 아닌, 저런 타이포그래픽으로 모션 그래픽을 연출하고 싶어"

**해석:**
- 사이트 전체 브랜드 톤 = Nexora(이미지 3) 결. 다크는 핵심 아니고, **비대칭 그리드 + 큰 타이포 + 색면 + 그래픽 일러스트 + 미니멀 애니메이션** 부분.
- 헤로 = 사진 X. **타이포·색면·미니멀 모션 그래픽으로 구성**. 동네 풍경 사진은 헤로 아닌 다른 곳(타운 페이지의 본문 섹션 등)에서 쓸 수 있음.
- "미니멀 애니메이션" = framer-motion·CSS 트랜지션 정도. 무거운 비디오·3D X. 텍스트가 들어오는 부드러운 페이드·슬라이드·색면이 회전하는 정도.

## 4. Popin 적용 가이드 (어디에 어떤 톤)

레퍼런스 톤을 Popin의 화면별로 어떻게 분배할지.

### 4.1 헤로 / 랜딩 / 홈 → 포스터·매거진 톤 (이미지 3·5·9·10)
- 거대 한글 카피 ("당신의 아이디어가 머물 자리" 류)
- 색면 1~2개 + 작은 일러스트·타이포 모션 그래픽
- 영문 모노타입 캡션 ("POPUP SPACE / SHORT TERM")
- 사진 없음
- 미니멀 애니메이션: 색면 슬라이드 인, 텍스트 페이드, 일러스트 회전·점멸

### 4.2 타운 페이지 → 포스터 시리즈 (이미지 9·5)
- 헤로: 오렌지 또는 크림 색면 + 거대 한글 "성수" + 작은 영문 "SEONGSU / POPUP TOWN NO. 01" + 미니멀 일러스트 (지도·아이콘)
- 본문은 사진 사용 — **여기서 동네 분위기 실물 사진**이 자리 잡음 (당초 §6.2의 헤로 → 본문으로 위치 이동)
- 각 타운마다 다른 액센트 컬러 가능 — 성수=오렌지, 부암=녹색, 연남=핑크 등 (Phase 2 검토)

### 4.3 공간 상세 → 사진 중심 + 깔끔한 정보 (이미지 7 Appear Here 톤)
- 매물 사진이 절대 주인공. 그래픽·색면 최소화.
- 정보 위계는 Appear Here·직방 톤. 거대 타이포 안 씀.
- 검증 배지·가격은 작고 명확하게.

### 4.4 검색 결과 / 마이 → 실용 톤
- 정보 밀도 우선. 그래픽 톤 줄임.
- 카드는 비대칭 그리드(이미지 3) 응용 가능 — 큰 카드 1 + 작은 카드 2 패턴.

### 4.5 임대인 등록·관리 → 도구 톤
- 박 사장님 페르소나 — 그래픽 톤보다 **명료함**. 큰 글씨·큰 버튼·단순한 폼.
- 색면·일러스트 거의 안 씀.

→ 즉 **사이트 전체가 한 톤이 아니다.** 브랜드 톤(헤로·타운 매거진)은 그래픽 디자인 적, 기능 영역(매물·임대인)은 실용 적. 이게 정상이고, 직방·에어비앤비도 마케팅 페이지와 매물 페이지의 톤이 다름.

## 5. 잠정 토큰 방향

토큰 문서는 별도 산출물(다음 작업). 여기선 큰 방향만.

**색 (잠정):**
- 주 텍스트 — 검정 #0A0A0A 근사
- 배경 — 크림 #F4EFE6 또는 흰 (페이지에 따라)
- **액센트 메인 — 오렌지** (이미지 5·9의 결, "팝업" 활기와 결합)
- 보조 액센트 — 검정 색면 (이미지 5·9의 검정 포스터처럼)
- 그레이 — 5단계 (배경/구분선/placeholder/secondary/disabled)
- 시스템 — 성공 녹·경고 황·에러 적 (절제)

→ **결정 #2**: 기존 design_direction.md의 "파랑은 액센트로만"이 충돌. 메인 액센트를 오렌지로 바꿀지, 파랑 유지할지, 또는 둘 다 쓸지.

**타이포 (잠정):**
- 본문 — Pretendard (PRD 기존 확정)
- 거대 타이포 — Pretendard 800 또는 ExtraBold. 또는 더 굵은 디스플레이용 폰트 1개 추가(예: Pretendard Black, 또는 GT America, 또는 무료 대안)
- 모노 캡션 — JetBrains Mono 또는 IBM Plex Mono (영문 인덱스 라벨용)

→ **결정 #3**: 디스플레이용 폰트를 Pretendard 800으로 갈지, 별도 폰트 추가할지.

**모션 (잠정):**
- framer-motion 또는 CSS transition
- 페이지 진입 시 색면 슬라이드 인 + 텍스트 페이드
- 헤로 일러스트 미니멀 루프 (회전·점멸·shape morph)
- 호버 — 색면 살짝 이동, 카드 살짝 떠오름
- **장식적 큰 모션 금지** (memory/design_direction.md 기존 원칙 유지)

## 6. 안티-가드 (안 갈 방향)

레퍼런스에서 의도적으로 추출하지 않은 부분 + AI 디자인 함정.

- 다크 모드 메인 ✕ (이미지 2·3의 다크는 영감만, Popin은 밝은 톤 기본)
- 글래스모피즘·블러 효과 ✕ (이미지 2 톤 거절)
- 그라데이션 남발 ✕
- 둥근 라운드 16px 이상 무차별 ✕ — 라운드는 카드 한 종(예: 24px)만, 작은 칩은 8px
- 동네 풍경 사진 헤로 ✕ (사용자 명시 거절)
- 이모지 헤로 카피 ✕
- 모든 섹션 가운데 정렬 ✕ — 좌하단·우하단·좌상단 의도적 비대칭 활용 (이미지 8·9·10)
- "SaaS틱한 푸른 글로우" ✕
- 영문 일변도 ✕ — 한국 사용자라 **한글이 주연, 영문은 캡션**

## 7. 결정 사항 (확정)

토큰 작업의 기준선. 시안 보고 흔들리면 여기로 돌아와 재결정.

### #1. 이미지 1 (Popin 시안) → **참고용, 좋은 부분만 추출**
- 사용자 답 — "No preference" (특정 의도 없음)
- 처리 — 헤로 비주얼·동네 풍경 사진 헤로는 폐기. 다음만 유지: 헤로 카피 톤 ("당신의 아이디어가 머물 자리"), 하단 타운 라인 ("지금 매장이 열리는 동네: 망원·성수·연남·…"), 헤더 우측 KO/EN 토글 + 검정 회원가입 패턴.

### #2. 메인 액센트 컬러 → **오렌지** ✅
- 이미지 5·9 결. 활기·팝업 톤.
- 사용 규칙 — 메인 CTA·강조·브랜드 시그널만. 정보·링크 텍스트는 검정/회색. 한 화면에 오렌지 면적이 너무 크면 부담스러우니 절제.
- 기존 design_direction.md "파랑은 액센트로만" 원칙은 폐기 (다음 작업에서 memory 갱신).

### #3. 디스플레이용 폰트 → **Pretendard Black/ExtraBold 한 종** ✅
- 폰트 1종 운영 — 본문 Regular/Medium, 강조 SemiBold/Bold, 디스플레이 ExtraBold/Black.
- 영문 모노 캡션용 보조 폰트 1종은 추가 — JetBrains Mono (무료, 가볍게 임포트).

### #4. 타운 페이지 헤로 → **타이포·색면 (사진은 본문)** (잠정, 시안 보고 재확인)
- 사용자 의도(헤로는 사진 X, 타이포 모션 그래픽)와 일관성 우선.
- 동네 풍경 사진은 타운 페이지 **"동네 분위기" 본문 섹션**으로 이동. 헤로에서 본문으로의 위치 이동만.
- 부수 효과 — spec.md §14 #2(헤로 이미지 7장 확보) 부담이 줄어듦. 본문 사진은 갯수·비율 자유로워서 임시 픽 활용도 쉬움.

### #5. 한글 + 영문 모노 캡션 → **적용** ✅
- 패턴 — 큰 한글 타이포 위·아래에 작은 영문 모노 ("성수동 팝업타운 / SEONGSU · POPUP TOWN NO. 01")
- 적용 범위 — 헤로, 타운 페이지, 주요 섹션 헤더. 공간 상세·문의 폼·임대인 등록 같은 기능 영역에는 안 씀 (실용 톤).
- 영문은 자동 번역 X — 운영자가 직접 잡음 (카피 톤 일관성). 일단 슬러그(`seongsu`)를 대문자로 쓰는 형태 권장.
