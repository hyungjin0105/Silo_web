## 4. 사진 가이드 (시안용 placeholder)

PRD §4.4 — **5장 / 4종 의무**. 시안 단계는 실제 매물 사진 확보 전이라 임시 픽 사용.

### 4.1 임시 픽 옵션

**A) Unsplash 검색어 (사용자가 픽해서 URL 교체)**

| 슬롯 | 종류 | 추천 검색어 |
|---|---|---|
| 1 | 외관 | `korean storefront ground floor brick`, `seoul small shop street` |
| 2 | 내부 전경 | `minimal cafe interior wide angle`, `empty retail space natural light` |
| 3 | 내부 다른 각도 | `cafe corner window`, `small shop counter view` |
| 4 | 화장실/주방 | `small kitchen cafe`, `industrial bathroom minimal` |
| 5 | 자유 | `cafe street view seoul`, `popup store interior` |

**B) 시안용 placeholder 색면 (사진 픽 전 임시)**

각 슬롯에 회색 면 + 라벨 — 예:

```
┌────────────────────┐
│  외관              │   배경 #E8E4DD (color-gray-100)
│                    │   라벨 color-gray-500
│  16:9              │
└────────────────────┘
```

→ 권장 — **B로 시작**. 사진 확보 후 A로 교체. 시안에서 사진 톤이 너무 좋으면 디자인 평가가 사진에 묻힘.

### 4.2 사진 슬롯별 비율

- 사진 #1 (외관) — 16:9 또는 4:3
- 사진 #2~4 — 4:3 또는 정사각형
- 사진 #5 (자유) — 비율 자유

캐러셀에서는 모두 같은 비율 (4:3 권장)로 통일 표시.

## 5. 상태별 분기 시드

시안에 한 화면 안에서 **3가지 상태**를 같이 보여달라고 요청. 같은 매물의 다른 사용자 시야.

### 상태 A — 일반 조회 (찜 X)

- 우상단 ♡ (회색, 빈 하트)
- 하단 고정 CTA "문의하기"
- 가용 캘린더에서 11월 첫 주는 가능(하늘색), 11월 둘째 주는 협의 중(노랑)

### 상태 B — 찜한 상태

- 우상단 ♥ (오렌지 채워진 하트) — `color-accent`
- 카드 상단에 작은 칩 "♥ 찜한 공간" (선택, 디자인 따라)
- 나머지는 상태 A와 동일

### 상태 C — 미검증 공간 (다른 매물)

스펙 차이만 명시. 시안에선 공간 시드를 한 줄로 갈음.

```
공간 ID         #1088
공간명          망원동 8평 1층 골목 가게
검증 배지       미검증 (회색)
조회수          23
받은 문의       0
사진            등록 직후라 사진 4장만 등록 (1장 부족 = 검증 대기 사유)
```

- 검증 배지 칩 = 회색 (`color-gray-100` 배경, `color-gray-500` 텍스트)
- 카드 또는 상세 상단에 안내 띠 — "이 공간은 검증 대기 중입니다. 며칠 내 검증이 완료돼요."
- 문의 CTA는 활성 (미검증이라도 문의 가능, 노출 하위라 검색에서 적게 보일 뿐)

## 7. 시드 재사용 가능성

- **공간 상세 시안** — 이 시드 그대로
- **검색 결과 카드** — §1 기본·검증 배지·가격만 발췌
- **타운 페이지 매물 리스트** — 동일
- **문의 폼 시안** — §3 임차인 시드 + 공간 미니 카드 (§1 발췌)
- **임대인 받은 문의 시안** — §3 임차인 자기소개 + §1 공간

같은 시드를 여러 화면에서 재사용 — 시안 전체에 일관된 "한 명의 임차인이 한 공간에 문의하는" 흐름이 만들어짐. 일관성이 디자인 평가의 명료함을 높임.
